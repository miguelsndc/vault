If $S$ and $T$ are [[Sets]], then their intersection $S\cap T$, is a set that consists of all elements that **are in both** $S$ and $T$.

Let $S = \{2, 3, 4, 5, 7, 9\}$
Let $T = \{1, 2, 4, 7, 9, 10\}$

Therefore, the set $S\cap T = \{2, 4,7,9\}$. o
