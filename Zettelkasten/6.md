Seja $L$ uma linguagem regular. Por definição existe um AFD $M = (Q, \Sigma, \delta, q_{0}, F)$ que reconhece $L$.

Vamos construir uma GLC $G = (V, \Sigma, R, S)$ tail que a linguagem gerada pela gramática seja exatamente a linguagem reconhecida pelo autômato, ou seja, $L(G) = L(M)$.

Mapeamos os componentes de $M$ para $G$ da seguinte forma:

- Variáveis: Para cada estado do autômato, criamos uma variável na gramática: $V = Q$
- Terminais: O alfabeto da gramática é o mesmo do autômato.
- Variável Inicial: O estado inicial do autômato torna-se a variável inicial da gramática. Logo $S = q_{0}$.
- Regras de derivação: as regras são criadas com base nas transições e nos estados de aceitação de $M$:
	1. Para cada transição do autômato onde do estado $q_{i}$ lendo o símbolo $a$ vai para o estado $q_{j}$, (ou seja, $\delta(q_{i}, a) = q_{j}$), adicionamos a regra:
	$$\begin{align*}
q_{i}\rightarrow aq_{j}
\end{align*}$$
	2. Para cada estado de aceitação $q_f \in F$, adicionamos a regra de produção:
	$$\begin{align*}
q_{j}\rightarrow \epsilon
\end{align*}$$

Precisamos também mostrar que o autômato aceita uma palavra $w$ se e somente se a gramática deriva essa mesma palavra $w$.
