Seja $L$ uma linguagem regular. Por definição existe um AFD $M = (Q, \Sigma, \delta, q_{0}, F)$ que reconhece $L$.

Vamos construir uma GLC $G = (V, \Sigma, R, S)$ tail que a linguagem gerada pela gramática seja exatamente a linguagem reconhecida pelo autômato, ou seja, $L(G) = L(M)$.

Mapeamos os componentes de $M$ para $G$ da seguinte forma:

- Variáveis: Para cada estado do autômato, criamos uma variável na gramática: $V = Q$
- Terminais: O alfabeto da gramática é o mesmo do autômato.
- Variável Inicial: O estado inicial do autômato torna-se a variável inicial da gramática. Logo $S = q_{0}$.
- Regras de derivação: as regras são criadas com base nas transições e nos estados de aceitação de $M$:
	1. Para cada transição do autômato onde do estado $q_{i}$ lendo o símbolo $a$ vai para o estado $q_{j}$, (ou seja, $\delta(q_{i}, a) = q_{j}$), adicionamos a regra:
	$$\begin{align*}
q_{i}\rightarrow aq_{j}
\end{align*}$$
	2. Para cada estado de aceitação $q_f \in F$, adicionamos a regra de produção:
	$$\begin{align*}
q_{j}\rightarrow \epsilon
\end{align*}$$

Precisamos também mostrar que o autômato aceita uma palavra $w$ se e somente se a gramática deriva essa mesma palavra $w$.

Seja $w = a_{1}a_{2}\cdots a_{n}$ uma palavra aceita por $M$, isso significa que existe um caminho de transições no AFD:
$$\begin{align*}
q_{0}\xrightarrow{a_{1}}q_{1}\xrightarrow{a_{2}}q_{2}\cdots\xrightarrow{a_{n}}q_{f}
\end{align*}$$
onde $q_{f} \in F$

Pela nossa construção de $G$, cada passo desse caminho corresponde exatamente a uma aplicação de regra de produção. A derivação em mais à esquerda em$G$ será:
$$\begin{align*}
q_{0}\rightarrow a_{1}q_{1}\rightarrow a_{1}a_{2} q_{2}\cdots\rightarrow a_{1}a_{2}\cdots a_{n}q_{f}
\end{align*}$$
Como $q_{f}$ é um estado de aceitação em $M$, nós criamos a regra $q_{f} \rightarrow \epsilon$ em $G$. Aplicando essa regra no final da derivação temos:

$$\begin{align*}
a_{1}a_{2}\cdots a_{n} q_{f} \rightarrow a_{1}a_{2}\cdots a_{n} \epsilon \rightarrow a_{1} a_{2}\cdots a_{n} &= w
\end{align*}$$
Mostramos que o processo de processar uma string em um AFD é equivalente ao processo de derivar uma string na GLC construída. Como podemos construir uma GLC equivalente para **qualquer** AFD existente, concluímos que toda linguagem regular é também livre de contexto.
