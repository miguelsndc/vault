A Piecewise [[Functions|function]] is a special type of function whose rules depend of the input. See:
$$
f(x) =
\begin{cases}
x^{2} &\text{  if  } x < 0; \\
x+1 &\text{  if  } x \ge 0;
\end{cases}$$
